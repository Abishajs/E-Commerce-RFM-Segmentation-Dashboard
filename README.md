# E-Commerce Customer Behavior Analysis & RFM Segmentation Dashboard
## Project Overview
Developed an interactive Power BI dashboard to analyze customer behavior, revenue trends, product performance, and sales insights using MySQL data.
## Tools Used
- Power BI
- MySQL
- DAX
- Power Query
## Key Features
- RFM Customer Segmentation
- Customer Retention Analysis
- Revenue Trend Analysis
- Geographic Sales Analysis
- Executive KPI Dashboard
- Strategic Business Recommendations
## Customer Segments
- Champions
- Loyal Customers
- Potential Customers
- New Customers
- At Risk Customers
- Lost Customers
## Dashboard Pages
1. Executive Overview
2. Customer Segmentation Analysis
3. Product Performance Analysis
## Business Insights
- Identified high-value customer segments.
- Highlighted seasonal revenue patterns.
- Recommended targeted retention strategies.
- Provided actionable business recommendations.
